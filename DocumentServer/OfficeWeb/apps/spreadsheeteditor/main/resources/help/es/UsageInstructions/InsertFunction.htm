Inserte funciones

La capacidad de realizar cálculos básicos es un razón principal para usar una hoja de cálculo. Para hacerlo, usted puede insertar una fórmula necesaria a mano usando operadores matemáticas comunes o insertar una fórmula predefinida - Función.

Para insertar una función,

  1. seleccione la celda a donde usted quiere insertar una función,
  2. pulse el icono Insertar función que se sitúa en la barra de herramientas superior,
    o haga el clic con el botón derecha en la celda seleccionada y seleccione la opción Insertar función del menú,
    o pulse el icono que se sitúa en la barra de fórmulas,
  3. seleccione la función necesaria en la lista,
  4. introduzca argumentos de la función a mano o seleccione un rango de celdas que usted quiere usar como los argumentos.

Aquí hay las funciones disponibles agrupadas por categorías:

Categoría de función Descripción Funciones
Funciones de fecha y hora Se usan para mostrar correctamente fecha y hora en su hoja de cálculo. DATE; DAY; HOUR; MINUTE; MONTH; NOW; SECOND; TIME; TODAY; YEAR
Funciones de información Se usan para darle información sobre los datos en la celda seleccionada o en un rango de celdas. ERROR.TYPE; ISBLANK; ISERR; ISERROR; ISEVEN; ISLOGICAL; ISNA; ISNONTEXT; ISNUMBER; ISODD; ISREF; ISTEXT; N; NA; TYPE
Funciones lógicas Se usan sólo para darle respuesta si la función devuelve TRUE o FALSE. AND; FALSE; IF; IFERROR; NOT; OR; TRUE
Funciones de búsqueda y referencia Se usan para fácilmente encontrar la información en la lista de datos. CHOOSE; HLOOKUP; LOOKUP; VLOOKUP
Funciones de matemática y trigonometría Se usan para realizar las operaciones de matemáticas y trigonometría básicas como adición, multiplicación, división, redondeamiento etc. ABS; ACOS; ASIN; ATAN; ATAN2; CEILING; COS; DEGREES; EVEN; EXP; FLOOR; INT; LN; LOG; LOG10; MOD; MROUND; ODD; PI; POWER; PRODUCT; RADIANS; RAND; ROUND; ROUNDDOWN; ROUNDUP; SIGN; SIN; SQRT; SQRTPI; SUM; TAN
Funciones estadísticas Se usan para analizar los datos: encontrar valor promedio, valores más grandes o más pequeños en un rango de celdas. AVERAGE; COUNT; MAX; MAXA; MIN; MINA
Funciones de texto y datos Se usan para mostrar correctamente datos de texto en su hoja de cálculo. CONCATENATE; EXACT; LOWER; REPT; T; UPPER
