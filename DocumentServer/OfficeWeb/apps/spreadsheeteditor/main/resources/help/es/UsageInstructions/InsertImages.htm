Inserte imágenes

Para insertar una imagen en la hoja de cálculo,

  1. mete el cursor a un lugar donde se quiere que aparezca la imagen,
  2. Pulse el icono Insertar imagen en la barra de herramientas superior,
  3. seleccione una de las opciones siguientes para subir la imagen:
    • la opción Imagen desde archivo abrirá la ventana de diálogo Windows estándar para selección de archivo. Navegue el disco duro de su ordenador en búsqueda del archivo necesario y pulse el botón Abrir
    • la opción Imagen desde URL abrirá la ventana donde usted puede introducir la dirección web de un imagen necesario, después pulse el botón OK
  4. cuando el imagen ha sido añadido ya usted puede cambiar su tamaño y posición.

    Para cambiar el tamaño de la imagen, arraste pequeños círculos y cuadrados situados en los bordes de la imagen. Para cambiar la posición de la imagen, use el icono que aparece cuando usted mantiene el cursor del ratón encima del gráfico.

If you have several objects (charts or images) overlapped you can arrange their order clicking the necessary object with the mouse and selecting the desired arrangement type from the right-click menu:

  • Traer al primer plano - para mover el objeto seleccionado al primer plano,
  • Enviar al fondo - para mover el objeto seleccionado detrás de otros objetos,
  • Mover adelante - para mover el objeto seleccionado adelante por un nivel en lo relativo a otros objetos,
  • Mover atrás - para mover el objeto seleccionado atrás por un nivel en lo relativo a otros objetos.

Para borrar la imagen insertada, haga el clic en la imagen y pulse el tecla Delete.
