Cambie formato de número

Usted puede cambiar el formato de número, i.e. la aparencia de los números introducidos en la celda. Para hacerlo,

  1. seleccione una celda, un rango de celdas usando el ratón o toda la hoja de cálculo pulsando la combinación de los teclas Ctrl+A,
  2. pulse el icono Formato de número que se sitúa en la barra de herramientas superior y seleccione el formato de número deseado:
    • General - se usa para mostrar datos introducidos como los números sin formato de un modo compacto sin ningunos signos adicionales,
    • Número - se usa para mostrar números con 0-30 dígitos después de un punto decimal donde un separador de miles es añadido entre cada grupo de tres dígitos antes del punto decimal,
    • Entero - se usa para mostrar números como un entero,
    • Exponencial - se usa para mantener números cortos convirtiendo en una cadena de tipo d.dddE+ddd o d.dddE-ddd donde cada d es un dígito de 0 a 9,
    • Moneda - se usa para incluir uno de los símbolos de moneda siguientes: $ Dólar, € Euro, £ Libra, p. Rublo, ¥ Yen, y dos posiciones decimales,
    • Fecha - se usa para mostrar fechas. Usted puede seleccionar uno de los formatos de fecha siguientes: MM-dd-aa, MM-dd-aaaa, dd-MM-aa, dd-MM-aaaa, dd-MMM-aaaa, dd-MMM, MMM-aa,
    • Hora - se usa para mostrar hora. Usted puede seleccionar uno de los formatos de hora siguientes: HH:mi, HH:MI:ss, hh:mi tt, hh:mi:ss tt,
    • Porcentaje - se usa para mostrar datos como un porcentaje acompañado por el signo %,
    • Texto - se usa para mostrar valores numéricos como un texto sin formato con una precisión alta.
  3. Cambie el número de posiciones decimales, si es necesario:
    • use el icono Aumentar decimales que se sitúa en la barra de herramientas superior para mostrar más dígitos después del punto decimal,
    • use el icono Disminuir decimales que se sitúa en la barra de herramientas superior para mostrar menos dígitos después del punto decimal.
