View file information

To access the detailed information about the currently edited spreadsheet, click the File icon at the left sidebar and select the Spreadsheet Info option.

The file information includes spreadsheet title, author, location, creation date, persons with the rights to view or edit the spreadsheet, and statistics: the number of pages, paragraphs, words, symbols, symbols with spaces.

To close the File pane and return to your spreadsheet, select the Return to Spreadsheet option.
