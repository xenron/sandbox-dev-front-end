Tastaturkürzel

Arbeit mit der Kalkulationstabelle
Kalkulationstabelle drucken Strg+P Druckt die Tabelle mit einem der verfügbaren Drucker oder speichert in eine Datei.
Suchen Strg+F Öffnet das Fenster Suchen, um die Suche nach einer Zelle mit dem gesuchten Zeichen durchzuführen.
Herunterladen als Strg+Umschalt+S Öffnet die Leiste Herunterladen als, um die aktuelle Kalkulationstabelle auf die Festplatte Ihres Rechners in eines der unterstützten Formate zu speichern: XLSX, XLS, ODS, CSV, HTML.
Navigation
Zum Anfang der Zeile springen Pos 1 Outline the A column of the current row.
Zum Anfang der Kalkulationstabelle springen Strg+Pos 1 Outline the A1 cell.
Zum Ende der Zeile springen Ende Outline the last cell of the current row.
Zum Ende der Kalkulationstabelle springen Strg+Ende Outline the bottom right cell of your spreadsheet.
Move to the previous sheet Alt+PgDn Move to the previous sheet in your spreadsheet.
Move to the next sheet Alt+PgUp Move to the next sheet in your spreadsheet.
Move up one row Up Arrow Outline the cell above the current one in the same column.
Move down one row Down Arrow Outline the cell below the current one in the same column.
Move left one column Left Arrow, or
Tab		 Outline the previous cell of the current row.
Move right one column Right Arrow, or
Tab+Shift		 Outline the next cell of the current row.
Data Selection
Select all Ctrl+A, or
Ctrl+Shift+Spacebar		 Select the entire worksheet.
Select column Ctrl+Spacebar Select an entire column in a worksheet.
Select row Shift+Spacebar Select an entire row in a worksheet.
Select fragment Shift+Arrow Select the cell by cell.
Select from cursor to beginning of row Shift+Home Select a fragment from the cursor to the beginning of the current row.
Select from cursor to end of row Shift+End Select a fragment from the cursor to the end of the current row.
Extend the selection Ctrl+Shift+Home Select a fragment from the current selected cells to to the beginning of the worksheet.
Undo and Redo
Undo Ctrl+Z Reverse the latest performed action.
Redo Ctrl+Y Repeat the latest undone action.
Cut, Copy, and Paste
Cut Ctrl+X Cut the the selected data and send them to the computer clipboard memory. The cut data can be later inserted to another place in the same worksheet, into another spreadsheet, or into some other program.
Copy Ctrl+C Send the selected data to the computer clipboard memory. The copied data can be later inserted to another place in the same worksheet, into another spreadsheet, or into some other program.
Paste Ctrl+V Insert the previously copied/cut data from the computer clipboard memory to the current cursor position. The data can be previously copied from the same worksheet, from another spreadsheet, or from some other program.
Data Formatting
Bold Ctrl+B Make the font of the selected text fragment bold giving it more weight or remove bold formatting.
Italic Ctrl+I Make the font of the selected text fragment italicized giving it some right side tilt or remove italic formatting.
Underline Ctrl+U Make the selected text fragment underlined with the line going under the letters or remove underlining.
Dateneingabe
Complete cell entry and move down Eingabetaste Complete a cell entry in the selected cell or the formula bar, and move to the cell below.
Complete cell entry and move up Umschalt+Eingabetaste Complete a cell entry in the selected cell, and move to the cell above.
Start new line Alt+Eingabetaste Start a new line in the same cell.
Cancel Escape Cancel an entry in the selected cell or the formula bar.
Delete to the left Backspace Delete one character to the left in the formula bar or in the selected cell when the cell editing mode is activated.
Delete to the right Entf Delete one character to the right in the formula bar or in the selected cell when the cell editing mode is activated.
Clear cell content Entf Remove the content (data and formulas) from selected cells without affecting cell format or comments.
