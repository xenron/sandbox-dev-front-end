Cut/copy/paste data

To cut, copy and paste data of your worksheet use the keybord,

  • Cut - select data and use the Ctrl+X key combination to delete the selected data and send them to the computer clipboard memory. The cut data can be later inserted to another place in the same worksheet, into another spreadsheet, or into some other program.

  • Copy - select data and use the Ctrl+C key combination to send the selected data to the computer clipboard memory. The copied data can be later inserted to another place in the same worksheet, into another spreadsheet, or into some other program.

  • Paste - select a place and use the Ctrl+V key combination to insert the previously copied/cut data from the computer clipboard memory to the current cursor position. The data can be previously copied from the same worksheet, from another spreadsheet, or from some other program.
