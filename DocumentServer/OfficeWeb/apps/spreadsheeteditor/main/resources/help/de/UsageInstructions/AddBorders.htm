Add borders

To add and format borders to a worksheet,

  1. select a cell, a range of cells with the mouse or the whole worksheet by pressing the Ctrl+A key combination,
  2. click the Add Borders icon situated at the top toolbar,
  3. select the border style you wish to apply: Outside Borders , All Borders , Top Borders , Bottom Borders , Left Borders , Right Borders , No Borders , Inside Borders , Inside Vertical Borders , Inside Horizontal Borders ;
  4. set the Border Width and select a Border Color clicking the corresponding option and selecting the color you need from the palette.
