Create a new spreadsheet or open an existing one

When TeamLab Spreadsheet Editor is open to create a new spreadsheet:

  1. click the File icon at the left sidebar,
  2. select the Create New option.

After you finished working at one spreadsheet, you can immediately proceed to an already existing spreadsheet that you have recently edited, or return to the list of existing ones.

To open a recently edited spreadsheet within TeamLab Spreadsheet Editor,

  1. click the File icon at the left sidebar,
  2. select the Open Recent option,
  3. choose the spreadsheet you need from the list of recently edited spreadsheets.

To return to the list of existing spreadsheets, click the Back icon at the top toolbar or the File icon at the left sidebar and select the Back option.
