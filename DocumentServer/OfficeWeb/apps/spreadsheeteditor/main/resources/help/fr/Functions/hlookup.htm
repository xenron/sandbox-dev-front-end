Fonction HLOOKUP

La fonction HLOOKUP est l'une des fonctions de recherche et de référence. Elle est utilisée pour effectuer la recherche horizontale pour une valeur dans la ligne supérieure d'un tableau ou renvoyer la valeur dans la même colonne basée sur un numéro d'index de rangée spécifiée.

La syntaxe de la fonction HLOOKUP est :

HLOOKUP (lookup-value, table-array, row-index-num[, [range-lookup-flag]])

où lookup-value est une valeur, table-array sont deux ou plusieurs lignes contenant des données classées par ordre croissant, row-index-num est un numéro de ligne de la même colonne de table-array,

range-lookup-flag est un argument optionnel. C'est une valeur logique : TRUE (vrai) ou FALSE (faux). Entrez FALSE (faux) pour trouver une correspondance exacte. Entrez TRUE (vrai) pour trouver une correspondance approximative, dans ce cas s'il n'y a pas de valeur qui correspond strictement à lookup-value, puis la fonction va choisir la plus grande valeur prochaine inférieure à lookup-value. Si cet argument est absent, la fonction trouve une correspondance approximative.

Remarque : si row-index-num est inférieur à 1, la fonction renvoie l'erreur #VALUE!. Si row-index-num est supérieur au nombre de lignes dans table-array, la fonction renvoie l'erreur #REF!. Si range-lookup-flag est mis à FALSE (faux), mais aucune correspondance n'est trouvée, puis la fonction renvoie l'erreur #N/A.

Pour appliquer la fonction HLOOKUP,

  1. sélectionnez la cellule dans laquelle vous souhaitez afficher le résultat,
  2. cliquez sur l'icône Insérer une fonction située sur la barre d'outils supérieure,
    ou cliquez avec le bouton droit sur la cellule sélectionnée et sélectionnez l'option Insérer une fonction depuis le menu,
    ou cliquez sur l'icône sutuée sur la barre de formule,
  3. sélectionnez la groupe fonctions Recherche et référence depuis la liste,
  4. cliquez sur la fonction HLOOKUP,
  5. saisissez les arguments nécessaires en les séparant par des virgules,
  6. appuyez sur la touche Entrée.

Le résultat sera affiché dans la cellule sélectionnée.
