Modifier le format de nombre

Vous pouvez facilement modifier le format de nombre, c'est à dire l’apparence d’un nombre entré dans votre feuille de caclul. Pour le faire,

  1. sélectionnez une cellule, une plage de cellules avec la souris ou la feuille de calcul entière en utilisant la combinaison de touches Ctrl+A,
  2. cliquez sur l'icône Format de nombre sutuée sur la barre d'outils supérieure et sélectionnez le format de nombre à appliquer:
    • Général - s'utilise pour afficher les données saisies en tant que nombres normaux de façon la plus compacte, sans aucun signe supplémentaire,
    • Nombre - s'utilise pour afficher les nombres avec 0-30 chiffres après le point décimal où un séparateur de milliers est ajouté entre chaque groupe de trois chiffres avant le point décimal,
    • Entier - s'utilise pour afficher les nombres sous la forme d'un nombre entier,
    • Scientifique (exponentiel) - s'utilise pour exprimer le nombre sous la forme d.dddE+ddd ou d.dddE-ddd où chaque d c'est un chiffre 0 à 9,
    • Devise - s'utilise pour inclure un des symboles monétaires suivants: $ Dollar, € Euro, £ Livre, p. Rouble, ¥ Yen, et deux décimales,
    • Date - s'utilise pour afficher les dates. Vous pouvez sélectionner l'un des formats de date suivants: MM-dd-yy, MM-dd-yyyy, dd-MM-yy, dd-MM-yyyy, dd-MMM-yyyy, dd-MMM, MMM-yy,
    • Heure - s'utilise pour afficher l'heure. Vous pouvez sélectionner l'un des formats d'heure suivants: HH:mm, HH:MM:ss, hh:mm tt, hh:mm:ss tt,
    • Pourcentage - s'utilise pour afficher les données en pourcentage avec un symbole de pourcentage %,
    • Texte - s'utilise pour afficher les valeurs numériques qui sont traitées comme du texte simple avec la plus grande précision disponible.
  3. modifiez le nombre de décimales, si nécessaire:
    • utilisez l'icône Ajouter une décimale sutuée sur la barre d'outils supérieure pour afficher plus de chiffres après le signe décimal,
    • utilisez l'icône Réduire les décimales sutuée sur la barre d'outils supérieure pour afficher moins de chiffres après le signe décimal.
