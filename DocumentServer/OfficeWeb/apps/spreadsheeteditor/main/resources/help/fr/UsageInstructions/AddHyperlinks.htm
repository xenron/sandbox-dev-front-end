Ajouter des liens hypertextes

Pour ajouter un lien hypertexte,

  1. sélectionnez une cellule dans laquelle un lien hypertexte sera ajouté,
  2. cliquez sur l'icône Ajouter un lien hypertexte sur la barre d'outils supérieure,
  3. après quoi une fenêtre apparaît dans laquelle vous pouvez spécifier les paramètres du lien hypertexte :
    • Type de lien - sélectionnez un type de lien que vous souhaitez insérer depuis la liste déroulante :

      Utilisez l'option Lien externe et entrez une URL au format http://www.example.com dans le champ Lien vers si vous avez besoin d'ajouter un lien hypertexte menant vers un site externe.

      Utilisez l'option Plage de données interne et sélectionnez une feuille et une plage dans les champs ci-dessous si vous avez besoin d'ajouter un lien hypertexte menant à une plage de cellules dans le même classeur.

    • Afficher - entrez un texte qui sera cliquable et mène vers l'adresse indiquée dans le champ supérieur.

      Remarque : si la cellule sélectionnée contient déjà des données, elles seront automatiquement affichées dans ce champ.

    • Texte de l'infobulle - entrez un texte qui sera visible dans une petite fenêtre contextuelle offrant une courte note ou étiquette lorsque vous placez le curseur sur un lien hypertexte.

  4. cliquez sur le bouton OK.

Lorsque vous placez le curseur sur le lien hypertexte ajouté, le texte de l'infobulle apparaîtra contenant le texte que vous avez spécifié. Pour suivre le lien cliquez dessus.

Pour supprimer un lien hypertexte ajouté, sélectionnez la cellule qui contient le lien hypertexte ajouté en utilisant le clavier et appuyez sur la touche Supprimer, ou cliquez à l'aide du bouton droit de la souris sur la cellule et sélectionnez l'option Effacer tout depuis la liste déroulante.
