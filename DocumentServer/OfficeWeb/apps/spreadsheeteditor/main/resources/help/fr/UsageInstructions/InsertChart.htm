Insérer un graphique

Pour insérer le graphique dans une feuille de calcul,

  1. sélectionnez la plage de cellules qui contiennent les données que vous souhaitez utiliser pour le graphique,
  2. cliquez sur l'icône Insérer un graphique sur la barre d'outils supérieure,
  3. après une fenêtre apparaît dans laquelle vous pouvez spécifier les paramètres du graphique :
    • Type et style de graphique - sélectionnez un type de graphique que vous souhaitez insérer : Ligne, Colonne, Barre, Zone, Secteur, Point, Stock et son style.
    • Plage de données - sélectionnez la plage de cellules, modifiez-la, et si nécessaire, choisissez la façon d'organiser les données. Vous pouvez aussi sélectionner Série en ligne ou Série en colonne à utiliser sur l'axe X .
    • Éléments de graphique - spécifiez le Titre du graphique et les éléments à afficher : Valeurs du graphique, Axes et Quadrillage ainsi que des titres des axes dans les champs correspondants.
    • Légende de graphique - spécifiez si vous souhaitez afficher une légende pour votre graphique ou pas. Si vous souhaitez qu'une légende s'affiche, laissez la case correspondante cochée et indiquez la position de votre légende à l'égard de votre graphique en sélectionnant l'option nécessaire à partir de la liste déroulante ci-dessous : Bas, Haut, Groit, Gauche.

  4. une fois que le graphique est ajouté, vous pouvez modifier sa taille et sa position.

    Pour modifier la taille du graphique, faites glisser de petits cercles et des carrés situés sur les bords du graphique. Pour modifier la position du graphique, utilisez l'icône qui apparaît lorsque vous placez le curseur de votre souris sur le graphique.

Si vous avez besoin de modifier le graphique inséré,

  1. sélectionnez le graphique avec la souris,
  2. cliquez sur l'icône Insérer un graphique sur la barre d'outils supérieure,
  3. faites toutes les modifications nécessaires dans la fenêtre Paramètres du graphique,
  4. cliquez sur le bouton OK pour appliquer les modifications et fermer la fenêtre.

Si vous avez plusieurs objets (graphiques ou images) qui se chevauchent vous pouvez arranger leur ordre en cliquant sur l'objet nécessaire avec la souris et en sélectionnant le type de disposition souhaitée depuis le menu contextuel :

  • Mettre au premier plan - pour déplacer l'objet sélectionné devant tous les autres objets,
  • Envoyer à fond - pour déplacer l'objet sélectionné derrière tous les autres objets,
  • Déplacer vers l'avant - pour déplacer l'objet sélectionné d'un niveau avant par rapport à d'autres objets,
  • Déplacer vers l'arrière - pour déplacer l'objet sélectionné au niveau supérieur par rapport à d'autres objets.

Pour supprimer le graphique inséré, cliquez dessus et appuyez sur une touche Supprimer.
