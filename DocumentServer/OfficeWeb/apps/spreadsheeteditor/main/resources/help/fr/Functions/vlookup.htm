Fonction VLOOKUP

La fonction VLOOKUP est l'une des fonctions de recherche et de référence. Elle est utilisée pour effectuer la recherche vertical d'une valeur dans la première colonne à gauche d'un tableau et retourner la valeur sur la même ligne basée sur un numéro d'index de colonne spécifiée.

La syntaxe de la fonction VLOOKUP est :

VLOOKUP (lookup-value, table-array, col-index-num[, [range-lookup-flag]])

où lookup-value est une valeur à rechercher, table-array deux ou plusieurs colonnes contenant des données triées dans l'ordre croissant, col-index-num est un numéro de colonne dans table-array, range-lookup-flag est une valeur logique TRUE (vrai) ou FALSE (faux),

range-lookup-flag est un argument optionnel. Entrez FALSE (faux) pour trouver une correspondance exacte. Entrez TRUE (vrai) pour trouver une correspondance approximative, dans ce cas, s'il n'y a pas de valeur qui correspond strictement à lookup-value, puis la fonction va choisir la plus grande valeur prochaine inférieure à lookup-value. Si cet argument est absent, la fonction trouve une correspondance approximative.

Remarque : si col-index-num est inférieur à 1, la fonction renvoie l'erreur #VALUE!. Si col-index-num est supérieur au nombre de colonnes dans table-array, la fonction renvoie l'erreur #REF!. Si range-lookup-flag est mis à FALSE (faux), mais aucune correspondance n'est trouvée, puis la fonction renvoie l'erreur #N/A.

Pour appliquer la fonction VLOOKUP,

  1. sélectionnez la cellule dans laquelle vous souhaitez afficher le résultat,
  2. cliquez sur l'icône Insérer une fonction située sur la barre d'outils supérieure,
    ou cliquez avec le bouton droit sur la cellule sélectionnée et sélectionnez l'option Insérer une fonction depuis le menu,
    ou cliquez sur l'icône sutuée sur la barre de formule,
  3. sélectionnez la groupe fonctions Recherche et référence depuis la liste,
  4. cliquez sur l'icône VLOOKUP,
  5. saisissez les arguments nécessaires en les séparant par des virgules,
  6. appuyez sur la touche Entrée.

Le résultat sera affiché dans la cellule sélectionnée.
