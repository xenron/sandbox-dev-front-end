Insérer une fonction

La possibilité d'accomplir les opérations de calcul de base est la raison principale d'utiliser de la feuille de calcul. Pour le faire, vous pouvez insérer manuellement une formule nécessaire en utlisant des opérateurs mathématiques ou insérer une formule prédéfinie - Fonction.

Pour insérer une fonction,

  1. sélectionnez une cellule dans laquelle vous voulez entrer une fonction,
  2. cliquez sur l'icône Insérer une fonction située sur la barre d'outils supérieure,
    ou faites un clic droit sur une cellule sélectionnée et sélectionnez l'option Insérer une fonction depuis le menu,
    ou cliquez sur l'icône qui se trouve devant la barre de formule,
  3. sélectionnez la fonction nécessaire depuis la liste,
  4. entrez manuellement les arguments de la fonction ou faites glisser pour sélectionner une plage de cellules à inclure comme argument.

Catégorie de fonctions Description Fonctions
Fonctions de date et d'heure S'utilisent pour afficher correctement la date et l'heure dans votre feuille de calcul. DATE; DAY; HOUR; MINUTE; MONTH; NOW; SECOND; TIME; TODAY; YEAR
Fonctions d’information S'utilisent pour vous donner les informations sur les données de la cellule sélectionnée ou une plage de cellules. ERROR.TYPE; ISBLANK; ISERR; ISERROR; ISEVEN; ISLOGICAL; ISNA; ISNONTEXT; ISNUMBER; ISODD; ISREF; ISTEXT; N; NA; TYPE
Fonctions logiques S'utilisent uniquement pour vous donner une réponse VRAI ou FAUX. AND; FALSE; IF; IFERROR; NOT; OR; TRUE
Fonctions de recherche et de référence S'utilisent pour trouver facilement les informations dans la liste de données. CHOOSE; HLOOKUP; LOOKUP; VLOOKUP
Fonctions mathématiques et trigonométriques S'utilisent pour effectuer des opérations mathématiques et trigonométriques telles que l'ajout, la multiplication, la division, l'arrondissement, etc. ABS; ACOS; ASIN; ATAN; ATAN2; CEILING; COS; DEGREES; EVEN; EXP; FLOOR; INT; LN, LOG; LOG10; MOD; MROUND; ODD; PI; POWER; PRODUCT; RADIANS; RAND; ROUND; ROUNDDOWN; ROUNDUP; SIGN; SIN; SQRT; SQRTPI; SUM; TAN
Fonctions statistiques S'utilisent pour analyser les données : trouver la valeur moyenne, les valeurs plus grandes ou plus petites dans une plage de cellules. AVERAGE; COUNT; MAX; MAXA, MIN
Fonctions de données et texte S'utilisent pour afficher correctement les données de texte dans votre feuille de calcul. CONCATENATE; EXACT; LOWER; REPT; T; UPPER
