Fonction IF

La fonction IF est l'une des fonctions logiques. Elle renvoie une valeur si la condition que vous spécifiez est vrai ( TRUE ), et une autre valeur si cette valeur est faux ( FALSE ).

La syntaxe de la fonction IF est :

IF(logical_test, value_if_true, value_if_false)

où logical_test, value_if_true, value_if_false sont les données entrées manuellement ou incluses dans la cellule à laquelle il est fait référence.

Pour appliquer la fonction IF,

  1. sélectionnez la cellule dans laquelle vous souhaitez afficher le résultat,
  2. cliquez sur l'icône Insérer une fonction située sur la barre d'outils supérieure,
    ou cliquez avec le bouton droit sur la cellule sélectionnée et sélectionnez l'option Insérer une fonction depuis le menu,
    ou cliquez sur l'icône sutuée sur la barre de formule,
  3. sélectionnez la groupe fonctions Logique depuis la liste,
  4. cliquez sur la fonction IF,
  5. entrez les arguments requis en les séparant par des virgules,
  6. appuyez sur la touche Entrée.

Le résultat sera affiché dans la cellule sélectionnée.
