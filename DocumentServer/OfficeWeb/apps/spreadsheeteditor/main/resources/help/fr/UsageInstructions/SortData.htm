Trier et filtrer les données

Trier les données

Vous pouvez trier rapidement vos données dans une feuille de calcul en utilisant l'une des options disponibles :

  • Croissant s'utilise pour trier vos données dans l'ordre croissant - De A à Z par ordre alphabétique ou de plus petit au plus grand pour les données numériques.
  • Décroissant s'utilise pour trier vos données dans l'ordre décroissant - De Z à A par ordre alphabétique ou de plus grand au plus petit pour les données numériques.

Pour trier vos données,

  1. sélectionnez une plage de cellules à trier,
  2. cliquez sur l'icône Trier et filtrer située sur la barre d'outils supérieure ou utilisez l'option correspondante du menu contextuel,
  3. sélectionnez l'option de tri nécessaire.

Filtrer les données

Pour afficher uniquement les lignes qui répondent aux certains critères utilisez l'option Filtrer. Pour activer un filtre,

  1. sélectionnez une plage de cellules à filtrer,
  2. cliquez sur l'icône Trier et filtrer située sur la barre d'outils supérieure,
  3. sélectionnez l'option Filtre.

La flèche de déroulement apparaît dans la première cellule de chaque colonne de la plage de cellules sélectionnée. Cela signifie que le filtre est activé.

Pour appliquer un filtre, cliquez sur la flèche de déroulement . Dans la fenêtre Filtre ouverte réglez les paramètres du filtre en utilisant l'un des moyens suivants :

  • Sélectionnez les données à afficher

    Décochez les cases à côté des données à cochez. Pour votre facilité toutes les données dans la fenêtre Filtre sont triées dans l'ordre croissant. Pour faciliter le processus utilisez le champ de recherche. Saisissez votre requête, complète ou partielle, dans le champ et appuyez sur la touche Entrée. Pour vider le champ de recherche, utilisez l'icône .

    Si vous avez besoin vous pouvez définir l'ordre de tri des données à afficher en cliquant sur le bouton pour trier vos données par ordre croissant, ou en utilisant le bouton pour les trier par ordre décroissant.

  • Appliquer le filtre personnalisé

    Cliquez sur le bouton Personnalisé dans le coin supérieur droit de la fenêtre Filtre. Dans la fenêtre ouverte Personnalisé sélectionnez un des critères disponibles dans la liste déroulante supérieure et entrez la valeur nécessaire dans le champ à droite.

    Pour ajouter encore un critère, sélectionnez l'option Et si vous avez besoin des données pour satisfaire les deux critères ou utilisez l'option Ou si l'un ou deux critères peuvent être satisfaits. Ensuite, sélectionnez le deuxième critère dans la liste déroulante inférieure et entrez la valeur nécessaire à droite.

Le bouton Filtre apparaît dans la première cellule de la colonne. Cela signifie que le filtre est appliqué.

Mettre sous forme de modèle de tableau

Pour faciliter le travail avec vos données TeamLab Spreadsheet Editor vous permet d'appliquer un modèle de tableau à une plage de cellules sélectionnée en activant le filtre. Pour le faire,

  1. sélectionnez une plage de cellules à mettre en forme,
  2. cliquez sur l'icône Trier et filtrer située sur la barre d'outils supérieure,
  3. sélectionnez l'option Mettre sous forme de modèle de tableau,
  4. cliquez sur le modèle nécessaire dans la galerie.

Le modèle sera appliqué à la plage de cellules sélectionnée et vous pourrez modifier les en-têtes du tableau et appliquer le filtre pour travailler avec vos données.

Effacer le filtre

Pour effacer le filtre,

  1. sélectionnez la plage de cellules contenant les données filtrées,
  2. cliquez sur l'icône Trier et filtrer située sur la barre d'outils supérieure,
  3. sélectionnez l'option Filtrer.
