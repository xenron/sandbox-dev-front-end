Fonction LOOKUP

La fonction LOOKUP est l'une des fonctions de recherche et de référence. Elle est utilisée pour renvoyer une valeur à partir d'une plage sélectionnée (ligne ou colonne contenant les données dans l'ordre croissant).

La syntaxe de la fonction LOOKUP est :

LOOKUP(lookup-value, lookup-vector, result-vector)

où lookup-value (valeur à rechercher), lookup-vector (une rangée ou colonne contenant des données classées par ordre croissant), lookup-result (une rangée ou colonne de données qui est de la même taille que lookup-vector) sont les données entrées manuellement ou incluses dans la cellule à laquelle il est fait référence.

La fonction recherche lookup-value dans lookup-vector et renvoie la valeur à partir de la même position dans lookup-result.

Remarque : si lookup-value est plus petite que toutes les valeurs dans lookup-vector, la fonction renvoie l'erreur #N/A. S'il n'y a pas de valeur qui correspond strictement à lookup-value, la fonction choisit la plus grande valeur dans lookup-vector qui est inférieure ou égale à la valeur.

Pour appliquer la fonction LOOKUP,

  1. sélectionnez la cellule dans laquelle vous souhaitez afficher le résultat,
  2. cliquez sur l'icône Insérer une fonction située sur la barre d'outils supérieure,
    ou cliquez avec le bouton droit sur la cellule sélectionnée et sélectionnez l'option Insérer une fonction depuis le menu,
    ou cliquez sur l'icône sutuée sur la barre de formule,
  3. sélectionnez la groupe fonctions Recherche et référence depuis la liste,
  4. cliquez sur la fonction LOOKUP,
  5. saisissez les arguments nécessaires en les séparant par des virgules,
  6. appuyez sur la touche Entrée.

Le résultat sera affiché dans la cellule sélectionnée.
