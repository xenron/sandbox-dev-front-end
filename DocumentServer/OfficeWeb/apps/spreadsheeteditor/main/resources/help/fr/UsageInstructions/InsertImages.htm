Insérer des images

Pour insérer une image dans la feuille de calcul,

  1. placez le curseur là où vous voulez insérer une image,
  2. cliquez sur l'icône Insérer une image sur la barre d'outils supérieure,
  3. sélectionnez l'une des options suivantes pour charger l'image :
    • l'option Image à partir d'un fichier ouvre la fenêtre de dialogue Windows standard pour sélectionner le fichier. Sélectionnez le fichier nécessaire sur le disque dur de votre ordinateur et cliquez sur le bouton Ouvrir
    • l'option Image à partir d'une URL ouvre la fenêtre où vous pouvez entrer l'adresse web de l'image nécessaire et cliquez sur le bouton OK
  4. une fois l'image ajoutée, vous pouvez modifier sa taille et sa position.

    Pour modifier la taille de l'image, faites glisser de petits cercles et des carrés situés sur les bords de l'image. Pour modifier la position de l'image, utilisez l'icône qui apparaît lorsque vous placez le curseur de votre souris sur l'image.

Si vous avez plusieurs objets (graphiques ou images) qui se chevauchent vous pouvez arranger leur ordre en cliquant sur l'objet nécessaire avec la souris et en sélectionnant le type de disposition souhaitée depuis le menu contextuel :

  • Mettre au premier plan - pour déplacer l'objet sélectionné devant tous les autres objets,
  • Envoyer à fond - pour déplacer l'objet sélectionné derrière tous les autres objets,
  • Déplacer vers l'avant - pour déplacer l'objet sélectionné d'un niveau avant par rapport à d'autres objets,
  • Déplacer vers l'arrière - pour déplacer l'objet sélectionné au niveau supérieur par rapport à d'autres objets.

Pour supprimer l'image insérée, cliquez dessus et appuyez sur une touche Supprimer.
