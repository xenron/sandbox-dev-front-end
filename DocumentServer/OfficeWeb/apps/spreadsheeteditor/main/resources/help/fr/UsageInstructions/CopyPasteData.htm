Couper/copier/coller les données

Pour couper, copier et coller les données de votre classeur utilisez le menu contextuel ou les icônes correspondantes sur la barre d'outils supérieure,

  • Couper - sélectionnez les données et utilisez l'option Couper pour supprimer les données sélectionnées et les envoyer vers le presse-papiers. Les données coupées peuvent être insérées ensuite à un autre endroit dans la même feuille de calcul, dans un autre classeur, ou dans un autre programme.

  • Copier - sélectionnez les données et utilisez l'icône Copier sur la barre d'outils supérieure ou cliquez avec le bouton droit et sélectionnez l'option Copier depuis le menu pour envoyer les données sélectionnées vers le presse-papiers. Les données copiées peuvent être insérées ensuite à un autre endroit dans la même feuille de calcul, dans un autre classeur, ou dans un autre programme.

  • Coller - sélectionnez l'endroit et utilisez l'icône Coller sur la barre d'outils supérieure cliquez avec le bouton droit et sélectionnez l'option Coller pour insérer les données précédemment copiées/coupées depuis le presse-papiers à la position actuelle du curseur. Les données peuvent être copiées à partir de la même feuille de calcul, à partir d'un autre classeur, ou provenant d'un autre programme.

Pour des raisons de sécurité certains navigateurs ne permet pas d'accéder au presse-papiers de votre ordinateur. Ainsi, lorsque vous essayez de sélectionner l'une de ces trois options, le programme vous demandera d'utiliser les raccourcis clavier communs à tous les navigateurs :

Alternativement, vous pouvez effectuer les opérations nécessaires en utilisant les combinaisons de touches suivantes :

  • Ctrl+X combinaison de touches pour couper ;
  • Ctrl+C combinaison de touches pour copier ;
  • Ctrl+V combinaison de touches pour coller.
