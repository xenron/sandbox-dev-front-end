ASINH Function

The ASINH function is one of the math and trigonometry functions. It is used to return the inverse hyperbolic sine of a number.

The ASINH function syntax is:

ASINH(number)

where number is any numeric value entered manually or included into the cell you make reference to.

To apply the ASINH function,

  1. select the cell where you wish to display the result,
  2. click the Insert Function icon situated at the top toolbar,
    or right-click within a selected cell and select the Insert Function option from the menu,
    or click the icon situated at the formula bar,
  3. select the Math and trigonometry function group from the list,
  4. click the ASINH function,
  5. enter the required argument,
  6. press the Enter button.

The result will be displayed in the selected cell.
