COLUMNS Function

The COLUMNS function is one of the lookup and reference functions. It is used to return the number of columns in a cell reference.

The COLUMNS function syntax is:

COLUMNS(array)

where array is a reference to a range of cells.

To apply the COLUMNS function,

  1. select the cell where you wish to display the result,
  2. click the Insert Function icon situated at the top toolbar,
    or right-click within a selected cell and select the Insert Function option from the menu,
    or click the icon situated at the formula bar,
  3. select the Lookup and Reference function group from the list,
  4. click the COLUMNS function,
  5. select a range of cells with the mouse or enter it manually, like this A1:B2,
  6. press the Enter button.

The result will be displayed in the selected cell.
