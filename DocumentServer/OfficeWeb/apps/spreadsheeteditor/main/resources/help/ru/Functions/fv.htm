Функция FV

Функция FV - это одна из финансовых функций. Вычисляет будущую стоимость инвестиции исходя из заданной процентной ставки и постоянной периодичности платежей.

Синтаксис функции FV:

FV(rate, nper, pmt [, [pv] [,[type]]])

где

rate (ставка) - процентная ставка инвестиции.

nper - количество платежей.

pmt - сумма отдельного платежа.

pv - текущая сумма платежей. Необязательный аргумент. Если он опущен, аргументу pv присваивается значение 0.

type (тип) - срок выплаты. Необязательный аргумент. Если его значение равно 0 или он опущен, предполагается, что платеж должен быть произведен в конце периода. Если значение аргумента type равно 1, платеж должен быть произведен в начале периода.

Примечание: выплачиваемые денежные средства (например, сберегательные вклады) представляются отрицательными числами; получаемые денежные средства (например, дивиденды) представляются положительными числами. Единицы измерения аргументов rate (ставка) и nper (количество платежей) должны быть согласованы между собой: используйте N%/12 для аргумента rate (ставка) и N*12 для аргумента nper (количество платежей), если речь идет о ежемесячных платежах, N%/4 для аргумента rate (ставка) и N*4 для аргумента nper (количество платежей), если речь идет о ежеквартальных платежах, N% для аргумента rate (ставка) и N*12 для аргумента nper (количество платежей), если речь идет о ежегодных платежах.

Эти числовые значения можно ввести вручную или использовать в качестве аргументов ссылки на ячейки.

Чтобы применить функцию FV,

  1. выделите ячейку, в которой требуется отобразить результат,
  2. щелкните по значку Вставить функцию , расположенному на верхней панели инструментов,
    или щелкните правой кнопкой мыши по выделенной ячейке и выберите в меню команду Вставить функцию,
    или щелкните по значку перед строкой формул,
  3. выберите из списка группу функций Финансовые,
  4. щелкните по функции FV,
  5. введите требуемые аргументы, разделяя их запятыми,
  6. нажмите клавишу Enter.

Результат будет отображен в выбранной ячейке.
