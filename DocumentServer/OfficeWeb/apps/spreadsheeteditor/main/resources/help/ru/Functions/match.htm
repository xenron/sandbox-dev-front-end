Функция MATCH

Функция MATCH - это одна из поисковых функций. Возвращает относительное положение заданного элемента в диапазоне ячеек.

Синтаксис функции MATCH:

MATCH(lookup-value, lookup-array[ , [match-type]])

где

lookup-value - это значение для поиска в lookup-array. Оно может быть числовым, логическим, текстовым или представлять собой ссылку на ячейку.

lookup-array - это одна строка или один стобец для анализа.

match-type - это вид совпадения. Это необязательный аргумент. Он может принимать одно из следующих числовых значений:

Числовое значение Комментарий
1 или опущено Значения должны быть отсортированы в порядке возрастания. Если точное совпадение не найдено, функция возвращает наибольшее значение, не превышающее lookup-value.
0 Значения могут быть отсортированы в любом порядке. Если точное совпадение не найдено, функция возвращает ошибку #N/A.
-1 Значения должны быть отсортированы в порядке убывания. Если точное значение не найдено, функция возвращает наименьшее значение, превышающее lookup-value.

Чтобы применить функцию MATCH,

  1. выделите ячейку, в которой требуется отобразить результат,
  2. щелкните по значку Вставить функцию , расположенному на верхней панели инструментов,
    или щелкните правой кнопкой мыши по выделенной ячейке и выберите в меню команду Вставить функцию,
    или щелкните по значку перед строкой формул,
  3. выберите из списка группу функций Поиск и ссылки,
  4. щелкните по функции MATCH,
  5. введите требуемые аргументы, разделяя их запятыми, или выделите мышью диапазон ячеек,
  6. нажмите клавишу Enter.

Результат будет отображен в выбранной ячейке.
