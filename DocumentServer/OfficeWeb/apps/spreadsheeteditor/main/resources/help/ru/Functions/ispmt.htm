Функция ISPMT

Функция ISPMT - это одна из финансовых функций. Используется для вычисления процентов, выплачиваемых за определенный инвестиционный период, исходя из постоянной периодичности платежей.

Синтаксис функции ISPMT:

ISPMT(rate, per, nper, pv)

где

rate - это процентная ставка по инвестиции.

per - это период времени, за который требуется вычислить размер процентов. Этот аргумент может принимать значения от 1 до nper .

nper - это количество выплат.

pv - это текущая стоимость выплат.

Примечание: выплачиваемые денежные средства (например, сберегательные вклады) представляются отрицательными числами; получаемые денежные средства (например, дивиденды) представляются положительными числами. Единицы измерения аргументов rate (ставка) и nper (количество выплат) должны быть согласованы между собой: используйте N%/12 для аргумента rate (ставка) и N*12 для аргумента nper (количество выплат), если речь идет о ежемесячных платежах, N%/4 для аргумента rate (ставка) и N*4 для аргумента nper (количество выплат), если речь идет о ежеквартальных платежах, N% для аргумента rate (ставка) и N*12 для аргумента nper (количество выплат), если речь идет о ежегодных платежах.

Числовые значения могут быть введены вручную или находиться в ячейке, на которую дается ссылка.

Чтобы применить функцию ISPMT,

  1. выделите ячейку, в которой требуется отобразить результат,
  2. щелкните по значку Вставить функцию , расположенному на верхней панели инструментов,
    или щелкните правой кнопкой мыши по выделенной ячейке и выберите в меню команду Вставить функцию,
    или щелкните по значку перед строкой формул,
  3. выберите из списка группу функций Финансовые,
  4. щелкните по функции ISPMT,
  5. введите требуемые аргументы, разделяя их запятыми,
  6. нажмите клавишу Enter.

Результат будет отображен в выделенной ячейке.
