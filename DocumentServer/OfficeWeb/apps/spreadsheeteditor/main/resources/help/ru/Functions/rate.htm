Функция RATE

Функция RATE - это одна из финансовых функций. Используется для вычисления размера процентной ставки по инвестиции исходя из постоянной периодичности платежей.

Синтаксис функции RATE:

RATE(nper, pmt, pv [, [[fv] [,[[type] [,[guess]]]]]])

где

nper - это количество выплат.

pmt - это сумма отдельного платежа.

pv - это текущая стоимость выплат.

fv - это значение будущей стоимости, то есть средства, оставшиеся после последней выплаты. Это необязательный аргумент. Если он опущен, аргумент fv полагается равным 0.

type - это срок выплат. Это необязательный аргумент. Если он равен 0 или опущен, предполагается, что платеж должен быть произведен в конце периода. Если аргумент type равен 1, платеж должен быть произведен в начале периода.

guess - это приблизительная оценка будущей ставки. Это необязательный аргумент. Если он опущен, аргумент guess полагается равным 10%.

Примечание: выплачиваемые денежные средства (например, сберегательные вклады) представляются отрицательными числами; получаемые денежные средства (например, дивиденды) представляются положительными числами. Единицы измерения аргументов guess (прогноз) и nper (количество платежей) должны быть согласованы между собой: используйте N%/12 для аргумента guess (прогноз) и N*12 для аргумента nper (количество платежей), если речь идет о ежемесячных платежах, N%/4 для аргумента guess (прогноз) и N*4 для аргумента nper (количество платежей), если речь идет о ежеквартальных платежах, N% для аргумента guess (прогноз) и N*12 для аргумента nper (количество платежей), если речь идет о ежегодных платежах.

Числовые значения могут быть введены вручную или находиться в ячейке, на которую дается ссылка.

Чтобы применить функцию RATE,

  1. выделите ячейку, в которой требуется отобразить результат,
  2. щелкните по значку Вставить функцию , расположенному на верхней панели инструментов,
    или щелкните правой кнопкой мыши по выделенной ячейке и выберите в меню команду Вставить функцию,
    или щелкните по значку перед строкой формул,
  3. выберите из списка группу функций Финансовые,
  4. щелкните по функции RATE,
  5. введите требуемые аргументы, разделяя их запятыми,
  6. нажмите клавишу Enter.

Результат будет отображен в выбранной ячейке.
