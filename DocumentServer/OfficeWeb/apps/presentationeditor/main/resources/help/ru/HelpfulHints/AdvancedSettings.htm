Дополнительные параметры редактора презентаций

Вы можете изменить дополнительные параметры онлайн-редактора презентаций. Для перехода к ним щелкните по значку Файл на левой боковой панели и выберите опцию Дополнительные параметры.... Можно также использовать значок , расположенный в правом верхнем углу верхней панели инструментов.

Доступны следующие дополнительные параметры:

  • Альтернативный ввод - используется для включения/отключения иероглифов.
  • Стандартное значение масштаба - используется для установки стандартного значения масштаба путем его выбора из списка доступных вариантов от 50% до 200%.
  • Отображать изменения при совместной работе - используется для определения изменений, которые необходимо подсвечивать во время совместного редактирования:
    • При выборе опции Все будут подсвечиваться все изменения, внесенные за время текущей сессии.
    • При выборе опции Последние будут подсвечиваться только те изменения, которые были внесены с момента, когда Вы последний раз нажимали на значок Сохранить .
  • Автосохранение - используется для включения/отключения опции автоматического сохранения изменений, внесенных при редактировании.
  • Направляющие выравнивания - используется для включения/отключения направляющих выравнивания, которые появляются при перемещении объектов и позволяют точно расположить их на слайде.
  • Единица измерения - используется для определения единиц, которые должны использоваться на линейках и в окнах свойств для измерения параметров элементов, таких как ширина, высота, интервалы, поля и т.д. Можно выбрать опцию Сантиметр или Пункт.

Чтобы сохранить внесенные изменения, нажмите кнопку Применить.
