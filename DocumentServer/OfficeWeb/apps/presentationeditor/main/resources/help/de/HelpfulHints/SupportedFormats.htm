Unterstützte Formate elektronischer Präsentationen

Eine Präsentation ist ein Satz von Folien, die unterschiedliche Typen des Inhalts einschließen können, wie Bilder, Mediadateien, Text, Effekte usw. Der TeamLab Presentation Editor unterstützt die folgenden Formate:

Format Beschreibung Durchsehen Bearbeiten Herunterladen
PPTX Office Open XML Presentation
Gezipptes, XML-basiertes, von Microsoft entwickeltes Dateiformat zur Präsentation der Kalkulationstabellen, Diagramme, Präsentationen und Textverarbeitungsdokumente		 + + +
PPT Dateiformat, das in Microsoft PowerPoint verwendet wird + +
ODP OpenDocument Presentation
Dateiformat, das mit der Anwendung Impress erstellte Präsentationen darstellt, diese Anwendung ist ein Bestandteil des OpenOffice-Pakets		 + +
PDF Portable Document Format
Dateiformat, das Dokumente unabhängig vom ursprünglichen Anwendungsprogramm, Betriebssystem und von der Hardwareplattform originalgetreu weitergeben kann		 +
