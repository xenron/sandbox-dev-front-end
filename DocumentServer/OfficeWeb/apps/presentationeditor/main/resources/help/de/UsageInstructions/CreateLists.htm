Erstellen Sie die Listen

Um eine Liste in einem Textbereich zu erstellen, führen Sie die folgenden Schritte aus:

  1. Bewegen Sie den Cursor zur Stelle, wo Sie eine Liste beginnen möchten (das könnte eine neue Zeile sein oder der bereits eingegebene Text).
  2. Wählen Sie einen Listentyp:
    • Eine Aufzählungsliste mit den Markierungen wird mit einem Klick aufs Symbol Aufzählungszeichen auf der oberen Symbolleiste erstellt.
    • Eine Nummerierungsliste mit den Zahlen oder Buchstaben wird mit einem Klick aufs Symbol Nummerierung auf der oberen Symbolleiste erstellt.

      Hinweis: Klicken Sie auf den schwarzen Pfeil neben dem Symbol Aufzählungszeichen oder Nummerierung, um zu wählen, wie die Liste aussehen wird.

  3. Ab jetzt wird jedes Mal, wenn Sie die Eingabetaste am Ende der Zeile drücken, ein neuer Eintrag in der Aufzählungs- oder Nummerierungsliste erscheinen. Um es zu beenden, drücken Sie die Rücktaste und fangen Sie einen üblichen Textabsatz an.
