Verwalten Sie die Folien

Standardmäßig hat eine gerade erstellte Präsentation eine leere Titelfolie.

Um eine neue Folie zu erstellen, führen Sie diese Schritte aus:

  1. Klicken Sie aufs Symbol Folie hinzufügen auf der oberen Symbolleiste.
  2. Wählen Sie eine Folie mit dem gewünschten Layout im Menü.

    Hinweis: Sie können das Layout der hinzugefügten Folie zu jeder Zeit ändern. Weitere Informationen dazu finden Sie in der Sektion Bestimmen Sie die Folienparameter.

Alternativ können Sie auf jede Folie mit der rechten Maustaste klicken und die Option Neue Folie im Kontextmenü wählen.

Eine neue Folie wird nach der gewählten Folie in die Liste der vorhandenen Folien links eingefügt.

Um eine Folie zu verdoppeln, führen Sie diese Schritte aus:

  1. Klicken Sie mit der rechten Maustaste auf die gewünschte Folie in der Liste der vorhandenen Folien links.
  2. Wählen Sie die Option Folie verdoppeln im Kontextmenü.

Die verdoppelte Folie wird nach der gewählten Folie in die Folienliste eingefügt.

Um eine Folie zu kopieren, führen Sie diese Schritte aus:

  1. Wählen Sie die gewünschte Folie in der Liste der vorhandenen Folien links.
  2. Drücken Sie die Tasten STRG+C.
  3. Wählen Sie eine Folie in der Liste, nach der die kopierte Folie eingefügt wird.
  4. Drücken Sie die Tasten STRG+V.

Um eine vorhandene Folie zu verschieben, führen Sie diese Schritte aus:

  1. Klicken Sie auf die gewünschte Folie in der Liste der vorhandenen Folien links.
  2. Ohne die Maustaste freizulassen, ziehen Sie den Mauszeiger zur gewünschten Stelle in der Liste (eine horizontale Linie zeigt eine neue Position).

Um die gewünschte Folie zu löschen, führen Sie diese Schritte aus:

  1. Klicken Sie mit der rechten Maustaste auf die unerwünschte Folie in der Liste der vorhandenen Folien links.
  2. Wählen Sie die Option Folie löschen im Kontextmenü.

Um alle vorhandenen Folien auszuwählen, führen Sie diese Schritte aus:

  1. Klicken Sie mit der rechten Maustaste auf eine beliebige Folie in der Liste der vorhandenen Folien links.
  2. Wählen Sie die Option Alle wählen im Kontextmenü.

Um mehrere Folien auszuwählen, führen Sie diese Schritte aus:

  1. Drücken Sie die Taste STRG und halten Sie diese gedrückt.
  2. Wählen Sie die gewünschten Folien in der Liste der vorhandenen Folien links.
