Füllen Sie die Objekte und wählen Sie die Farben

Sie können unterschiedliche Füllungen auf den Hintergrund der Folie sowie auf die AutoFormen anwenden.

Um die Füllung des Folienhintergrunds zu ändern, wählen Sie die gewünschten Folien in der Folienliste. Die Registerkarte Folieneinstellungen wird im rechten Seitenbereich geöffnet.

Um die Füllung einer AutoForm zu ändern, klicken Sie auf die gewünschte AutoForm. Die Registerkarte Formeinstellungen wird im rechten Seitenbereich geöffnet.

Bestimmen Sie den gewünschten Füllungstyp:

  • Farbfüllung - wählen Sie diese Option, um eine Farbe zu bestimmen, mit der der innere Bereich der gewählten AutoForm bzw. Folie gefüllt wird.

    Klicken Sie auf das Farbfeld unten und wählen Sie die gewünschte Farbe aus den vorhandenen oder geben Sie eine andere Farbe an:

    • Designfarben - die Farben, die dem gewählten Farbschema der Präsentation entsprechen.
    • Standardfarben - ein Satz von Standardfarben.
    • Benutzerdefinierte Farbe - klicken Sie auf diesen Link, wenn Sie keine passende Farbe auf der verfügbaren Palette finden. Klicken Sie irgendwo auf der geöffneten Palette und bestimmen Sie die gewünschte Sättigung (der sRGB-Farbwert wird oben angezeigt), dann klicken Sie auf den Button Hinzufügen:

      Hinweis: Dieselben Farbtypen können Sie bei der Auswahl der Strichfarbe für AutoFormen oder Schriftfarbe sehen.

  • Bild oder Textur - wählen Sie diese Option, um ein Bild oder eine vorgegebene Textur als Hintergrund für eine AutoForm bzw. Folie zu nutzen.
    • Wenn Sie ein Bild als Hintergrund für eine Form bzw. Folie verwenden möchten, können Sie ein Bild Aus Datei hinzufügen, indem Sie es auf der Festplatte Ihres Rechners wählen, oder Aus URL, indem Sie die entsprechende URL-Adresse ins geöffnete Fenster einfügen.
    • Wenn Sie eine Textur als Hintergrund für eine Form bzw. Folie nutzen möchten, öffnen Sie die Liste Aus Textur und wählen Sie die gewünschte Texturvoreinstellung.

      Momentan sind die folgenden Texturen vorhanden: Canvas, Carton, Dark Fabric, Grain, Granite, Grey Paper, Knit, Leather, Brown Paper, Papyrus, Wood.

    • Wenn das gewählte Bild kleiner oder größer als die AutoForm oder Folie ist, können Sie die Option Strecken oder Kachel aus dem Listenmenü wählen.

      Die Option Strecken erlaubt Ihnen die Größe des Bildes so anzupassen, damit es den kompletten Bereich der Folie bzw. AutoForm füllen könnte.

      Die Option Kachel erlaubt Ihnen nur einen Teil eines großen Bildes zu verwenden und die Originalgröße beizubehalten oder ein kleines Bild zu wiederholen und seine Originalgröße beizubehalten, dabei werden diese wiederholten Bilder den kompletten Bereich der AutoForm bzw. Folie füllen.

      Hinweis: Jede gewählte Textur-Voreinstellung füllt den kompletten Bereich, aber Sie können nach Bedarf den Effekt Strecken anwenden.

    • Für die AutoFormen (unabhängig vom gewählten Füllungstyp) können Sie auch die Undurchsichtigkeit einstellen, indem Sie den Schieberegler nutzen oder einen Prozentwert manuell eingeben.

      Der Standardwert ist 100%. Er entspricht der kompletten Undurchsichtigkeit. Der Wert 0% entspricht der kompletten Durchsichtigkeit.

  • Keine Füllung - wählen Sie diese Option, wenn Sie keine Füllung verwenden möchten.
