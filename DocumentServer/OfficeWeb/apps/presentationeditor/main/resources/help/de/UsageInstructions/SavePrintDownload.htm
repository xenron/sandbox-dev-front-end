Speichern/drucken/laden Sie Ihre Präsentation herunter

Um die aktuelle Präsentation zu speichern, führen Sie die folgenden Schritte aus:

  • Klicken Sie aufs Symbol Speichern auf der oberen Symbolleiste.
  • Oder nutzen Sie die Tastenkombination Strg+S.
  • Oder klicken Sie aufs Symbol Datei im linken Seitenbereich und wählen Sie die Option Speichern.

Um die aktuelle Präsentation zu drucken, führen Sie die folgenden Schritte aus:

  • Klicken Sie aufs Symbol Drucken auf der oberen Symbolleiste.
  • Oder nutzen Sie die Tastenkombination Strg+P.
  • Oder klicken Sie aufs Symbol Datei im linken Seitenbereich und wählen Sie die Option Drucken.

Danach wird eine PDF-Datei aufgrund der bearbeiteten Präsentation erstellt. Sie können diese öffnen und ausdrucken oder auf die Festplatte Ihres Rechners bzw. einen Datenträger speichern, um die Präsentation später auszudrucken.

Um die aktuelle Präsentation auf die Festplatte Ihres Rechners herunterzuladen, führen Sie die folgenden Schritte aus:

  1. Klicken Sie aufs Symbol Datei im linken Seitenbereich.
  2. Wählen Sie die Option Herunterladen als.
  3. Wählen Sie eines der verfügbaren Formate nach Ihrem Bedarf: PDF, PPTX.
