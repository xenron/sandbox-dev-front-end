Bestimmen Sie die Folienparameter

Um Ihre Präsentation zu personalisieren, können Sie ein Thema und Farbschema für die ganze Präsentation wählen sowie die Füllung des Hintergrunds oder das Layout für jede einzelne Folie ändern.

  • Um ein Thema für die Präsentation zu wählen, klicken Sie auf das gewünschte vorgegebene Thema auf der rechten Seite der oberen Symbolleiste. Das gewählte Thema wird auf alle Folien angewandt, wenn Sie vorher keine bestimmten Folien gewählt haben.
  • Um ein Farbschema für die Präsentation zu ändern, klicken Sie aufs Symbol Farbschema ändern auf der oberen Symbolleiste und wählen Sie das gewünschte Schema aus dem Listenmenü. Das gewählte Schema wird auf alle Folien angewandt, wenn Sie vorher keine bestimmten Folien gewählt haben.
  • Um die Füllung des Hintergrunds zu ändern:
    1. In der Folienliste links wählen Sie die Folien, auf die Sie die Füllung anwenden möchten. Oder klicken Sie an jeder leeren Stelle im Bearbeitungsbereich der aktuellen Folie, um den Füllungstyp für diese einzelne Folie zu ändern.
    2. Auf der Registerkarte Folieneinstellungen im rechten Seitenbereich wählen Sie die notwendige Option:
      • Farbfüllung - wählen Sie diese Option, um die gewünschte Farbe zu bestimmen, die auf die gewählten Folien angewandt wird.
      • Bild oder Textur - wählen Sie diese Option, um ein Bild oder eine vorgegebene Textur als Folienhintergrund festzulegen.
      • Keine Füllung - wählen Sie eine Option, wenn Sie keine Füllung verwenden möchten.

      Weitere Informationen über diese Optionen finden Sie in der Sektion Füllen Sie die Objekte und wählen Sie die Farben.

  • Um das Folienlayout zu ändern:
    1. In der Folienliste links wählen Sie die Folien, auf die Sie das neue Layout anwenden möchten.
    2. Klicken Sie aufs Symbol Folienlayout ändern auf der oberen Symbolleiste.
    3. Wählen Sie das gewünschte Layout im Menü.

      Alternativ können Sie mit der rechten Maustaste auf die gewünschte Folie in der Liste links klicken, die Option Layout ändern im Kontextmenü wählen und das gewünschte Layout bestimmen.

      Hinweis: Momentan sind die folgenden Layouts verfügbar: Title, Title and Object, Section Header, Two Objects, Two Texts and Two Objects, Title Only, Blank, Title, Object and Caption, Picture and Caption, Vertical Text, Vertical Title and Text.
