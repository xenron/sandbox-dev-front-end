Faster websites with RGraph

RGraph is a HTML5 Javascript charts library. Using the new HTML5 canvas tag, RGraph charts are created in the web browser, meaning quicker pages and less web server load. Using RGraph is an easy way of speeding up your website.

RGraph produces a wide variety of chart types - examples and documentation below.

Supported charts

RGraph supports over twenty different types of charts, including: Bar charts, Bi-polar charts, Donut charts, Fuel gauges, Funnel charts, Gantt charts, Gauge charts Horizontal Bar charts, Horizontal Progress bars, LED grids, Line charts, Meters, Odometers, Pie charts, Radar charts, Radial Scatter charts, Rose charts, Scatter charts, Thermometers, Vertical Progress bars and Waterfall charts. In addition there are variants available on many charts. There is also a growing range of animations and visual effects.
